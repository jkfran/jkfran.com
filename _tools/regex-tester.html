Regex quick reference

Characters

  • . — Any character
  • \d — Digit (0-9)
  • \w — Word char (a-z, 0-9, _)
  • \s — Whitespace
  • [abc] — Character set
  • [^abc] — Negated set

Quantifiers

  • * — 0 or more
  • + — 1 or more
  • ? — 0 or 1
  • {3} — Exactly 3
  • {2,5} — Between 2 and 5
  • {3,} — 3 or more

Anchors & Groups

  • ^ — Start of string/line
  • $ — End of string/line
  • \b — Word boundary
  • (abc) — Capture group
  • (?:abc) — Non-capturing group
  • a|b — Alternation (or)
